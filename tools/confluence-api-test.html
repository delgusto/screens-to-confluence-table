Confluence API permissions test

Tests whether your API token can do the three operations the plugin would need: find a space, create a page, upload an attachment, and update the page body.

What does this test do?
  1. GET the space by key — confirms auth + tells us the space ID
  2. POST a new page titled API Test — [timestamp]
  3. POST a 1×1 pixel PNG as an attachment on that page
  4. PUT an updated page body that references the attachment

Nothing is stored or transmitted anywhere except your Confluence instance. The credentials live only in this page's memory. If everything passes, click Delete test page to clean up.

Your API token stays in the current tab's memory and is never sent to any third party. Treat the token as a password and revoke it at id.atlassian.com after testing if you prefer.
